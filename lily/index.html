Getting Started

Included Files

After downloading all files and documentation you will have a .zip file available. Once you unzip this file you will find the following:

  • documentation
  • licensing
  • lily.zip
  • lily-child.zip
  • lily-psd.zip
  • lily-countdown-snippet.txt
  • lily.wordpress.yyyy-mm-dd.xml

The WordPress theme is labeled as lily.zip

Please note as further updates are made to the theme, the zip file will be suffixed with the corresponding version number (e.g. lily-2.0.zip).

WordPress Upload

To upload through your Dashboard, select Appearance → Themes → Add New → Upload Theme. Choose your theme zip file and select "Install Now". Once the theme is fully uploaded and installed, click "Activate" to activate the WordPress theme.

When uploading your theme with the installer, make sure you are uploading the theme .zip file, not the entire package you downloaded. This means you will be uploading lily.zip.

FTP Upload

To manually upload your new WordPress theme, login with your credentials to your website and locate the wp-content folder in your WordPress install files. Upload the un-zipped Lily folder into the: wp-content/themes folder.

Once uploaded, activate the theme by heading to Appearance → Themes menu in the WordPress Dashboard. Locate the Lily theme and hit "Activate".

Demo Content

You can install the theme demo content using the lily.wordpress.yyyy-mm-dd.xml file which comes included in the main download.

The first thing you will need to do is go to Tools → Import. Bear in mind if you have never done this before you will be asked to install an importer. Make sure you select the WordPress option.

After installing and activating the importer plugin you will now be able to upload the above .xml file. Just navigate to the same area as you did above and select the option you prefer.

You can either assign the demo content to an existing author or you can import the one included within the demo content itself - there is no right or wrong here so just choose whichever option you prefer.

Lastly, you will see another section labeled "Import Attachments". Within this, there is a checkbox which you can tick to Download and import file attachments.

Please note that this process can take some time to complete depending on the demo content file size which is being imported. In addition, please note that the uploaded attachments are placeholders only and not those found in the live preview.

How do I install the plugins?

Theme Customizer

Accessing the Customizer

Once you've activated the theme and installed all the necessary plugins, you should visit Appearance → Customize to setup the basic theme settings such as your logo, header images, slideshow, background, color palette and much more. Detailed instructions and tips can be found below.

Theme Settings

Site Title, Tagline & Logo: Set your site title, the tagline or upload a logo to replace default settings.

Colors: Change body text, accent, hover, header text, background, alpha transparency colors to any color of your choice. To do this, simply click on the "Colors" section and use the color picker to choose your preference. You can also change the alpha transparency opacity level where the property is used. All changes will update live within the preview.

Header Image: Lily uses this option to set the homepage header image. To set this up select an image from your library or upload a new image. For optimal resolution I would recommend to skip cropping and publish the image as is. You can also randomize this by adding multiple images to the collection, turn on the slideshow feature, as well as your speed preference.

Please make sure not to upload images which are very large as this will affect your page loading time. The recommended dimension is 1920 x 1080. I would also strongly recommend these images be losslessly compressed if possible.

Header Hero Text: Customize your hero text area to suit your needs. You can also add a call to action to whichever page you wish for your guests to visit, so easy.

Background Image: Upload a background image which is used across the whole webpage. Once uploaded you will be given additional options to select from such as background repeat, position and attachment.

Navigation: Select from your list of existing menus. Your selection will be added to the top site navigation area.

Widgets: You can add new widgets or remove existing ones from the available widget areas. More details of what these widgets are can be found in the Widgets section of the documentation.

Static Front Page: Set your front page and posts page from within the customizer.

How do I use the customizer?

Page Templates

Homepage Setup

The theme includes a custom homepage page template that allows you to have a completely widgetized homepage layout.

Simply create a new page and label it whatever you want. I called mine Homepage to keep things simple.

Next, select the Homepage page template option found in the Page Attributes meta box. Click on the Publish button and you're done.

Once you have followed the above steps, make sure you go Settings → Reading and select this same page as your Front page option. Your posts page can have an alternate page assigned to it as well.

Blog Setup

If you chose to setup your homepage to use the homepage template above, you'll need to also tell WordPress which page will display your posts.

To do this go to Pages → Add New to create a new page to be used as the blog page. Add a title to this page - in the live preview I have simply called this page Blog.

Once you have created the page and given it the name you can then publish it.

Please note, you do not need to add anything in the post editor as this page will only be created to let WordPress know what page to assign the blog posts to.

You can now go into Settings → Reading and assign the Posts page to the Blog page you created. Make sure you Save Changes once you are happy with your settings.

Timeline Page Setup

The theme includes a custom timeline page template that allows you to add your information in timeline format.

Watch this video to learn how to create a timeline page.

Contact Page Setup

The theme uses Contact Form 7 plugin for adding contact forms to your pages.

Simply create your form with the information you wish to capture, copy the shortcode and paste it either in a text widget or post editor.

For more information on how this plugin works make sure to visit the documentation page found here.

Widgets

Included Widgets

Lily comes with four widgetized locations, these include the homepage, homepage hero text, posts or pages widgets and a sidebar.

You can add widgets by visting Appearance → Widgets, or through the theme customizer.

In addition to the native WordPress widgets, the theme comes with the below custom widgets which you can drag and drop as needed.

  • ThemeChills Callout
  • ThemeChills Countdown Timer
  • ThemeChills Events
  • ThemeChills Map
  • ThemeChills Photos
  • ThemeChills Profiles
  • ThemeChills Text
  • Instagram Feed (ThemeChills)

Please note, that even though these widgets can be used site wide, the primary usage for them is to be used on your site homepage.

Instagram Feed Widget

To start using populating your Instagram feed on your website follow the steps below:

  • Visit the Instagram Feed WordPress plugin page and follow the installation steps.
  • Next, go to Appearance → Customize.
  • Once you're in the Customizer, click the Widgets → Homepage menu items.
  • Click on the Add a Widget button.
  • Under the "Search widgets..." text field type the "Insta" keyword.
  • Click on the Instagram Feed (ThemeChills) widget to add to your homepage widget area.
  • Add your required details to the widget.
  • Save and Publish once you're happy with your changes.

Please note, due to the June 1, 2016 Instagram policy changes for partner platforms you will have to upgrade to the Pro version of the plugin to show posts by Hashtag.
